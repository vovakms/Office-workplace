Для подключения к БД ЕИИС  требуется :

 

1)      Установить драйвер  HyTechODBC

В папке с программой открыть папку  «HyTechODBC» , запустить файл odbcinst.exe

 

2)      Добавление источника данных

В этой же папке запустить файл odbcad32.exe, 

откроется  «Администратор источников данных ODBC»,

переходим на закладку «Системный DSN»,

жмем кнопку «Добавить»
